How to use
----------
1) Open `index.html` in any modern browser.
2) Edit text directly in the HTML to customise for your CAS project.
3) All styles are in `styles.css` and scripts in `script.js`.
4) Assets are in the `assets/` folder. Replace `logo.svg` if you like.

Features
--------
- Sports-themed, mobile-responsive design
- Sections: Warm-Ups, Stretching & Recovery, Motivation, Training Basics, Feedback
- Accessible markup (landmarks, labels, alt text, colour contrast)
- Feedback form can download responses as a JSON file
- Email feedback via `mailto:` link
- No external libraries (fully offline)
